#### Goal

Build the core of a small React app that combines routing, forms, and data.

#### Steps

1. Complete what we built in lecture.
2. Build the main page or main view.
3. Add the form or input flow your app needs.
4. Connect the app to the data source used in class.
5. Make sure the core user flow is started before adding extras.

#### Extra practice if you want

- Add a second route.
- Add a simple detail view.
- Add starter content for one stretch feature.
