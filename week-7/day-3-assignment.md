#### Objective

Build a new React project that brings together the main Level 3 skills.

#### Requirements

1. Create a new repo for this project.
2. Use React with multiple components.
3. Include at least one route or clearly separated view.
4. Include at least one form or input feature.
5. Include at least one data feature such as create, read, update, or delete.
6. Make sure there is one clear user flow that can be demonstrated.
7. Include a short README that explains:
   - what the app does
   - how to run it
   - what features are working now

#### Submission

GitHub repo link

#### Rubric (20 pts)

- Meets the objective 0-5
- Main user flow works correctly 0-5
- React, forms, and data are used correctly 0-5
- Code is organized and readable 0-5
