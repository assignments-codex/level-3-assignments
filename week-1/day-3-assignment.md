#### Objective

Build a small Node.js project that shows you can initialize an app, use modules, and install a library.

#### Requirements

1. Create a Node.js project with a valid `package.json`.
2. Add at least one custom module and import it into your main file.
3. Install and use one npm library.
4. Add at least one working script in `package.json`.
5. When the app runs, it should log a small result to the terminal.
6. Include a short README that explains:
   - what the project does
   - how to install dependencies
   - how to run the project

#### Submission

GitHub repo link

#### Rubric (20 pts)

- Meets the objective 0-5
- Project runs correctly 0-5
- package.json and scripts are set up correctly 0-5
- Code is organized and readable 0-5
