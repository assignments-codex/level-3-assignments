#### Goal

Practice working with modules, package.json, and one installed library.

#### Steps

1. Complete what we built in lecture.
2. Create one file that exports a function.
3. Import that function into your main file and run it.
4. Install one small library with npm and use it in your project.
5. Add at least one script to `package.json` so your app can be started with npm.

#### Extra practice if you want

- Create a second function in another file and import both into your main file.
- Add a script called `dev` that runs the same entry file.
- Be ready to explain the difference between code you wrote and code you installed with npm.
