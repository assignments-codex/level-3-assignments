#### Goal

Use Node.js to read from and write to a text file, and practice simple user input with command line arguments.

#### Steps

1. Complete what we built in lecture.
2. Create a text file for your project to use.
3. Write a script that reads the file and prints its contents.
4. Write a script that adds new text to the file.
5. Use `process.argv` to pass in a value from the terminal.

#### Extra practice if you want

- Let the user choose between two commands such as `read` and `add`.
- Add the current date or time to each new line written to the file.
- Make a second text file and test your script with both files.
