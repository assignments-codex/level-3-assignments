#### Goal

Set up your first Node.js project and understand how Node is different from JavaScript in the browser.

#### Steps

1. Complete what we built in lecture.
2. Create a new folder for your project and run `npm init -y`.
3. Open `package.json` and identify the project name, version, and scripts section.
4. Create an entry file and run it with Node.
5. In your own words, be ready to explain these ideas:
   - Node runs JavaScript outside the browser
   - Node does not use the DOM like browser JavaScript
   - package.json keeps track of project information and scripts

#### Extra practice if you want

- Add a few `console.log()` statements that print strings, numbers, and variables.
- Change the project name in `package.json`.
- Add a script so you can run your file with `npm run start`.
