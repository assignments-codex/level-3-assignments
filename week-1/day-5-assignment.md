#### Objective

Build a simple Node.js command line app that stores and reads text from a file.

#### Requirements

1. Create a Node.js project with a valid `package.json`.
2. Your app must support these two actions:
   - add text to a file
   - read and display the file contents
3. Use `process.argv` for the user input.
4. Use Node.js file system features to read and write the file.
5. Add at least one script to `package.json` that helps run the app.
6. Include a short README that explains:
   - what the app does
   - how to install dependencies
   - how to run the commands

#### Submission

GitHub repo link

#### Rubric (20 pts)

- Meets the objective 0-5
- File reading and writing works correctly 0-5
- Command line input works correctly 0-5
- Code is organized and readable 0-5
