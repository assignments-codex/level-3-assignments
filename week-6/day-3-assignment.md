#### Objective

Build a React app that can display data and create new records.

#### Requirements

1. Use React components to organize the app.
2. Display data from your source on the page.
3. Add a controlled form for creating a new record.
4. Make sure a new record appears in the app after it is submitted.
5. Keep the user flow simple and easy to explain.
6. Include a short README that explains:
   - what the app does
   - how to run it
   - what data action is working
