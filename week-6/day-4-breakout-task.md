#### Goal

Practice updating and deleting data so the full CRUD pattern feels more familiar.

#### Steps

1. Complete what we built in lecture.
2. Add a way to edit an existing record.
3. Add a way to delete an existing record.
4. Make sure the page updates after each action.
5. Be ready to explain the full CRUD flow in your app.

#### Extra practice if you want

- Add a simple confirmation before delete.
- Use a separate edit form.
- Add a button to cancel editing.
