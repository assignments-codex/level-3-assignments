#### Goal

Set up Tailwind in a Vite project and apply utility classes to the page.

#### Steps

1. Complete what we built in lecture.
2. Add Tailwind to your Vite project and confirm the styles are loading.
3. Style at least one section of the page with Tailwind utility classes.
4. Extra practice if you want
   - add responsive classes for at least one breakpoint
   - add a hover or focus state to one element
