#### Goal

Practice props and `useState` in a very simple React app.

#### Steps

1. Complete what we built in lecture.
2. Create at least three components.
3. Pass props from a parent component into a child component.
4. Use `useState` in one component.
5. Add one button that changes text, a number, or what the user sees on the page.

#### Extra practice if you want

- Add a second prop to one component.
- Add a second button with a different state change.
- Create a small counter.
