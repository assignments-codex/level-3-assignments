#### Goal

Practice rendering data in React with lists, simple conditions, and controlled inputs.

#### Steps

1. Complete what we built in lecture.
2. Render a list from an array.
3. Show different content based on a condition.
4. Add one controlled input with `useState`.
5. Display the input value somewhere on the page.

#### Extra practice if you want

- Add a second input.
- Add a button that adds a new item to the list.
- Move the list into its own component.
