#### Goal

Add one useful feature that makes your app easier to use.

#### Steps

1. Complete what we built in lecture.
2. Add one feature such as search, filter, sort, tabs, or category buttons.
3. Make sure the feature changes what the user sees on the page.
4. Test the feature with different values.
5. Keep the final result simple and easy to explain.

#### Extra practice if you want

- Add a clear button.
- Combine search with a filter.
- Add a simple empty state message.
