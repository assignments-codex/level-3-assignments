#### Objective

Improve your app with one useful UI feature and cleaner component structure.

#### Requirements

1. Add one feature such as search, filter, sort, or tabs.
2. Make sure the feature works with your current app data.
3. Use props and components to keep the feature organized.
4. Keep the app simple, clear, and easy to demo.
5. Include a short README that explains:
   - what feature you added
   - how to test it
