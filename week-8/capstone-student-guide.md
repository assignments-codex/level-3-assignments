# Level 3 Capstone - Student Guide

## Objective

Build and present a small React app that shows what you learned in Level 3.

## What you are building

Your project should include:

- a React front end
- a clear user flow
- multiple components
- props and state
- at least one form or user input
- data that comes from an API, a database, or another source used in class

Keep the scope small. A simple app that works well is better than a large app that is unfinished.

## Minimum requirements

- public GitHub repo
- live URL if your instructor requires deployment
- at least two components
- at least one main user flow that works from start to finish
- one form or input feature
- one data feature such as fetch, create, read, update, or delete
- a README with setup steps and a short project description

## Presentation

Be ready to:

1. explain what your app does
2. demo the main user flow
3. show the main files in your project
4. explain how props, state, and data work in your app

## What to submit

- GitHub repo link
- live URL if required by your instructor
- README with:
  - project description
  - setup steps
  - how to use the app

## Tips

- Keep the design simple.
- Focus on making the main feature work.
- Make sure you understand your own code.
- Build a project you can explain clearly.
