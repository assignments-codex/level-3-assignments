# Level 3 Capstone - Interview Rubric

## Scoring (100 points)

### 1. Project demo - 40 points

- Main user flow works
- App is easy to follow
- Student can explain what the project does
- Project feels complete enough to present

### 2. Code walkthrough - 35 points

- Student can explain components, props, and state
- Student can explain forms, user input, and data flow
- Student can explain where the main features live in the project
- Student can answer basic questions about their own code

### 3. Project quality - 25 points

- Repo is organized
- README is present and useful
- App is stable enough to demo
- Scope is realistic for Level 3

## Evaluator notes

Use the capstone to focus on the student's project, explanation, and understanding.

A large live coding section is not required here because students have already completed badge interviews during the level.
