#### Goal

Add routing so your React app can have more than one page.

#### Steps

1. Complete what we built in lecture.
2. Add routing to your app.
3. Create at least two pages.
4. Add navigation so the user can move between pages.
5. Put your API feature on one page and different content on another page.

#### Extra practice if you want

- Add a third page.
- Add a not found page.
- Move the navigation into its own component.
