#### Goal

Fetch data in an effect and handle loading and error states cleanly.

#### Steps

1. Complete what we built in lecture.
2. In a page or component, use an effect to fetch JSON from a public API or a local mock.
3. Render three states: loading, error, and data.
4. Extra practice if you want
   - add a small transform (map, filter, or reduce) before rendering
   - cache the last successful result in state to avoid flashing on re-renders
